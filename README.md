# PINNs for High-Dimensional Option Pricing and SDE-Based Hedging

This repository implements Physics-Informed Neural Networks (PINNs) for high-dimensional option pricing and stochastic differential equation (SDE)-based hedging strategies.

## Features

- Modular object-oriented architecture
- PyTorch-based implementation
- PDE definitions for SDE-based models
- Collocation point sampling
- Model calibration
- Training utilities with TensorBoard support
- Hedging strategies
- Benchmarking tools
- Visualization utilities
- YAML configuration support
- Comprehensive logging
- Type hints and docstrings throughout

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/Yashu0508/PINNS-For_SDE-OptionPricing-.git
cd PINNS-For_SDE-OptionPricing-
pip install -r requirements.txt
```

## Project Structure

- `benchmarking/`: Benchmarking and evaluation tools
- `visualization/`: Plotting and visualization utilities
- `configs/`: YAML configuration files
